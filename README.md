# LOB_Analysis

## Backend Set up

### Configute DataBase

Congfigure your database connection in `./backend/db.json`

### Create Tables

Run `stock_list_loader.py` and `umap_data_loader.py` respectively to create basic table for the project.

### Run backend app

Run `app.py` file.

## Frontend Set up

### Frontend Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run serve
```
